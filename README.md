Basic beat detection & animation lab.

![demo_video](docs/demo.gif)


- [Install](###install)
- [Dev](###run-dev)

### Install

> install `pnpm` if not installed
```
npm i -g pnpm
```

```bash
pnpm i
```


### Run dev
```bash
pnpm dev
```


### Build
```bash
pnpm build
```
